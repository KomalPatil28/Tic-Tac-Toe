# Tic Tac Toe Game

Tic Tac Toe is a simple two-player game developed in Java. Players take turns marking X and O on a 3x3 grid, and the game automatically detects the winner or a draw.
This project demonstrates basic game logic and Object-Oriented Programming concepts.

## Features
- Two-player gameplay
- 3x3 game board
- Winner detection (rows, columns, diagonals)
- Draw detection
- Simple and interactive gameplay

## Tech Stack
- Java
- Object-Oriented Programming (OOP)
- Eclipse IDE

## Project Structure
miniProjects
│
├── src/ # Java source files
└── tictactoe.java


## How to Run
1. Clone or download this repository
2. Open the project in Eclipse
3. Go to the `src` folder
4. Run the Main.java file
5. Follow the instructions to play the game
